**Алгоритм Кристофидеса для TSP:**

1. **Строим MST** - минимальное остовное дерево на всех вершинах
2. **Находим нечетные вершины** - вершины с нечетной степенью в MST  
3. **Минимальное паросочетание** - находим паросочетание минимального веса на нечетных вершинах
4. **Объединяем MST + паросочетание** - получаем эйлеров граф
5. **Строим эйлеров цикл** - обходим все ребра без повторений
6. **Преобразуем в гамильтонов цикл** - пропускаем уже посещенные вершины

**Объяснение 1.5-аппроксимации:**
- Вес MST ≤ вес оптимального TSP (удаление одного ребра из TSP дает остовное дерево)
- Вес паросочетания ≤ 0.5×вес оптимального TSP (оптимальный TSP можно разбить на два паросочетания)
- Итого: MST + паросочетание ≤ 1.5×OPT
